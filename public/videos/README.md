# Videos Directory

This directory is for video files used in the application.

## Expected Files:
- `hero-money-abstract.mp4` - Used in the home page hero section

## Format Guidelines:
- Use MP4 format for maximum compatibility
- Optimize videos for web (compress to reasonable size)
- Recommended max file size: 10MB per video
- Use H.264 codec for best browser support

## Usage:
Videos are referenced in components like `VideoWallpaper` component.
Example: `/videos/hero-money-abstract.mp4`

## Note:
Large video files should be stored in a CDN or external service in production.
